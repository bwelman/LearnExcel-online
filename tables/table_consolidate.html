Task: Consolidating ranges

The summarizing (consolidating) of data in separate ranges to a new range.

Similar data in different ranges can be summarized (consolidated) into a new range. In practice, the separate ranges are often in separate worksheets and the merged range is placed in a new worksheet. When you consolidate the source data, you apply a summary function (sum, average, count, ...) to create the summary data. To consolidate ranges, the columns must have a heading, a label.

File: Consolidation.xlsx
  1. Open the practice file.
  2. Select cell A1 in the worksheet Total.
    The lists to summarize are in the worksheets PRA and PRB. The consolidated data will be put in worksheet Total.
  3. Choose Data > Consolidate (group Data Tools).
  4. Check that the function Sum is selected. If not, select this function.
  5. Put the cursor in the box Reference.
  6. Select worksheet PRA > range A1:B7 > Add.
  7. Select worksheet PRB > range A1:B7 > Add.
  8. Check Use labels in Top row and Left column
  9. Click OK.
    Attention: The aggregation is static. When the source data change, the consolidation results will not, then you should perform a new consolidation. Using pivot tables is often a better choice to summarize data.
