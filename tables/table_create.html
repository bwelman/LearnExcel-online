Task: Convert Range to Table

How to convert a range to an Excel table.

File: Candyboxes.xlsx
  1. Open the practice file.
  2. Select a cell within the data range.
  3. tab Insert > Table

    The dialog box Create Table appears. The address of the range to convert is already present.

    Important: If your range has labels that you want to use as column headers, check the option My table has headers. When there no labels, then uncheck this option. Excel adds automatically headers to each column with generic names: Column1, Column2, …
  4. Click OK.
    The range is converted to a table:
    • A table format (style) is applied.
    • Filter/Sorting buttons appear in each column heading.
    • The tab Design (Table Tools) appears, and here you can see that the table has been given a default name like Table1.
    Tip:

    It is wise to change the default name to a meaningful name so that you can work with the table more easily later. You do this by selecting the name and then typing in the new name.
