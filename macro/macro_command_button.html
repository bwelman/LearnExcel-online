Task: Creating a Command Button

How to create a command button to run a macro.

To perform this task it is necessary that the macro CellFormat was created.

A very convenient way to run a macro is by placing a command button on the worksheet.

  1. Choose Developer tab > Insert (group Controls).
  2. Click on Button (Form Control)
    The cursor will change to a +. With this cursor you can draw a rectangle in the worksheet for the button.
  3. Click the left mouse button and draw a rectangle with the desired dimensions for the button somewhere in the worksheet. Then release the left mouse button.
    The drawn rectangle will turn into a button with a text like Button 1.Furthermore you will see the dialog box Assign Macro.
  4. Select the macro CellFormat and click OK.
  5. Select the text on the button and change it to a more appropriate text, for example Cell format.
  6. Deselect the button by clicking anywhere in the worksheet.
  7. Test the macro on a cell with text via the command button.
