Task: Editing Macros

How to change the options and program code of a macro.

To perform this task, the macro CellFormat should have been created.

A macro can be changed without recording the macro again. Both the options of the macro as the program code can be viewed and modified.

Changes to be made: Assign the shortcut key Control t to the macro and change the font size to 18 pt.

  1. Choose Developer tab > Macros (group Code).
  2. Choose Options.
  3. Enter the letter t in the box for the shortcut key and change in the description 16pt in 18pt. Click on OK.
  4. Choose now Edit.

    A new window Microsoft Visual Basic will appear with the program code of the macro.

    Sub CellFormat()
'
' CellFormat Macro
' Format cell in Comic Sans, 16pt, bold, italic and red
'

'
    With Selection.Font
        .Name = "Comic Sans MS"
        .FontStyle = "Bold Italic"
        .Size = 16
        .Strikethrough = False
        .Superscript = False
        .Subscript = False
        .OutlineFont = False
        .Shadow = False
        .Underline = xlUnderlineStyleNone
        .Color = 255
        .TintAndShade = 0
        .ThemeFont = xlThemeFontNone
    End With
    Selection.Columns.AutoFit
End Sub
  5. You can find the number 16 in two places. Change it to 18 in both places.
  6. Close the window Microsoft Visual Basic with the close button in the upper right corner.
  7. Test the macro on a cell with text using the keyboard shortcut CTRL t.
