What is the Solver?

A brief description of the add-in Solver.

The Solver is an add-in for Excel to perform optimization calculations. In such calculations, you want to find the optimal value for a formula in one cell, mostly the minimum or maximum or a very specific value.

The cell you want to find the optimal value for is called the objective cell. The result of the formula in the objective cell always depends on values in other cells. When these values are adjusted, the value of the objective cell will also change. You can change the value of these cells to influence the result in the objective cell. These cells are called variable cells, changing cells, or decision variables.

It often happens that the variable cells can not have every possible value. Those values are often bound to certain limits. This can also be true for other cells that influence the value of the objective cell. So the value in cells can have limitations. Limitations are called restrictions or constraints. The possible solution can depend on restrictions or limits to the values of other cells. The Solver adjusts the values in the decision variable cells to satisfy the limits or constraint cells and produces the result you want for the objective cell.

In summary, you have:

  • One objective cell, which should get an optimal value (minimum, maximum, other).
  • One or more variable cells whose values can be adjusted to get a different result for the objective cell.
  • Possible restrictions on the values of some cells.

To be able to carry out such an optimization calculation, you must first set up a suitable calculation model in Excel. This is often much more difficult than using the Solver itself.
