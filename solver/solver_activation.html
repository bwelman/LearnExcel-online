Task: Enabling Solver

Instructions on how to check if the Solver is available in Excel, and if not how to enable the Solver.

The Solver is an add-in and isn't available by default through the menu of Excel. If the Solver is not present on the ribbon, then it has to be activated first. This is a one time action.

  1. Choose tab Data and check if the group Analyze exists and if so whether the Solver is present.
    Attention: Only proceed if the group Analyze with the Solver is not present.
  2. Choose File > Options > Add-Ins
    A list with Microsoft Office Add-ins is displayed.
  3. Select Excel Add-ins in the box Manage.
  4. Click on Go....
    A list of available Add-ins is displayed.
  5. Select the check box for Solver Add-in and click on OK.
  6. Repeat the check in the first step.
