Task: Creating New Workbook

Instructions on how to create a new workbook.

Any time you can create a new workbook. This can be a blank workbook, but also a new workbook based on a particular template.

  1. Click on File > New.
    In the right part a list with available templates is displayed.
  2. Click on Blank workbook.
    A new workbook with one worksheet is created.
