Task: Format with 2 conditions

Conditional formatting with 2 conditions.

In a spreadsheet, the realized turnover and the turnover to be achieved of the salesmen are tracked. Some of them have not met their target. The task is to apply the background for the turnover with one of the following colors: green if the target has been met and red if the target has not been met.

File: Turnover_Q1.xlsx
  1. Open the practice file.
  2. Select all cells with the turnover data.
  3. Choose Home > Conditional Formatting (Group Styles) > Manage Rules....
  4. Click on New Rule....
    The dialog box New Formatting Rule appears.
  5. Select Format only cells that contain, then specify Cell Value less than =B2 and choose a red fill color as Format.
  6. Click OK.
    The dialog box Conditional Formatting Rules Manager appears again.
  7. Click on New Rule and create a new rule on the same way with the following settings:
    The specified fill color is green.
  8. Click OK.
    In the manager screen now there are two rules:
  9. Click OK.
    The desired format will be applied now.
