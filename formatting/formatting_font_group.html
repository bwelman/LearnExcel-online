Font

How to change the format of cells through the commands in the font group.

The group Font (tab Home) has a number of commands that allows to change the appearance of text in cells. To apply these changes, you must first select the cell(s) and then use the command button. The options in this group are:

Font, choice list through down arrow.
Font size, choice list through down arrow.
Increase / Decrease Font size, always in steps of 2 points.
Style, successively: bold, italic, underlined. The specific underscore can be determined using the down arrow.
Font color, the color can be chosen through the down arrow.
Fill color, the color can be chosen through the down arrow.
