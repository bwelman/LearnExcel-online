Task: Exponential trend line

An exponential trend line is a curved trend line which is especially suitable for data sets whose values rise or fall faster and faster.
Attention: There should be no null values in the data set.

The equation of the trend line is y = a*eb*x where a and b are constants.

The following example is about the number of players in a tennis tournament that remains after each round. The tournament has been started with 128 players.

File: Trendlines.xlsx
  1. Open the practice file and select worksheet Exponential
  2. Select a cell in the data area.
  3. Insert an XY (Scatter) Chart.
  4. Select the data series in the chart by clicking on a point. Right mouse click and choose Add Trend line.
  5. Select at Trendline Options: Exponential. Let equation and R-squared value be displayed in the chart.
  6. Change the chart title in Number of players per round.
  7. Add Axis titles and change title Y-axis in Number of players and title X-axis in Round.
  8. Drag in the chart the equation to an empty place.

R2 = 1, which means that an exponential trend line fits the data perfect. This is because the values in this example are exact numbers.

Worksheet functions

Equation: y = a*e(b*x)
  • a =EXP(INDEX(LINEST(LN(known_ys) ; known_xs) ; 1 ; 2))
  • b =INDEX(LINEST(LN(known_ys) ; known_xs) ; 1)
