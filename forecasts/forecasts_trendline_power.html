Task: Power trend line

A power trend line is a curved line that is very suitable for collections of data that increase at a specific rate, for example the acceleration of a formula 1 car.
Attention: There should be no null values in the data set.

The equation of the trend line is: y = a*xb where a and b are constants.

The following example is about the distance of an accelerating race car in intervals of 1 second.

File: Trendlines.xlsx
  1. Open the practice file and select worksheet Power
  2. Select a cell in the data area.
  3. Insert a XY (Scatter) Chart.
  4. Select the data serie in the chart by clicking on a point. Right mouse click and choose Add Trend line.
  5. Select at Trendline Options: Macht. Let equation and R-squared value be displayed in the chart.
  6. Change the chart title in Distance.
  7. Add Axis titles and change title Y-axis in Distance (meter) and title X-axis in Time (sec).
  8. Drag in the chart the equation to an empty place.

R2 = 0,9987 which means that a power trend line fits the data very well.

Worksheet functions

Equation: y = a*xb
  • a =EXP(INDEX(LINEST(LN(known_ys) ; LN(known_xs)) ; 2))
  • b =INDEX(LINEST(LN(known_ys) ; LN(known_xs)) ; 1)
