Overview Lookup Functions

An overview of some functions for lookups.

HLOOKUP

This function looks up for a given value in the first row of a list (table), and returns the corresponding value from another row of the array.

HLOOKUP(lookup_value, table_array, row_index_num, [range_lookup])
lookup_value
Required. The value that you want to look for in the first row of the list.
table_array
Required. The list that you want to search.
row_index_num
Required. The row number from the list, that you want the corresponding value to be returned from. The first row has number 1.
range_lookup
Optional. Possible values: TRUE (for the closest match) or FALSE (for exact matching).

VLOOKUP

This function looks up for a given value in the first column of a list (table) and returns the corresponding value from another column of the array.

VLOOKUP(lookup_value, table_array, column_index_num, [range_lookup])
lookup_value
Required. The value that you want to look for, in the first column of the list.
table_array
Required. The list that you want to search.
column_index_num
Required. The column number from the list that you want the corresponding value to be returned from. The first column has number 1.
range_lookup
Optional. Possible values: TRUE (for the closest match) or FALSE (for exact matching).
