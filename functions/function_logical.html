Task: Logical Function IF

A task using conditional operations through the function IF.

Logical functions do something with the results TRUE or FALSE. The best known and most widely used logical function is the IF function. Simply said , the operation of this function does the following: if a certain condition is true, then do this, otherwise do that. The syntax of the formula is

=IF(condition,what to do if true,what to do if false)

The condition is a logical test with possible results TRUE or FALSE.

Tip: If need more than one condition, then you can use a new IF() function inside the first IF() function. Such structures become quickly complicated and difficult to read. Better make use of the logical functions AND(), OR() and NOT().
In the following figure you see some marks. Besides the marks the text sufficient or insufficient should be displayed. A result is insufficient if the mark is below 5.5.
  1. Start with a new worksheet.
  2. Enter the data you see in the figure.
  3. Select cell B2 and press the button Insert Function on the formula bar.
  4. Select in the Logical category the function IF and click OK.
    The Function Arguments dialog box appears.
  5. Put the cursor in the Logical_test box and enter A2<5.5.
  6. Put the cursor in the Value_if_true box and enter insufficient.
  7. Put the cursor in the Value_if_false box and enter sufficient.
    Attention: Excel automatically puts double quotes around texts.
  8. Click OK.
  9. Select cell B2 and drag the fill handle to B6.
