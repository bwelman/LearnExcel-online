Task: Calculating Annuities

Example of a calculation of an annuity with the repayment of a loan.

There are several ways to determine the repayment of a loan. The following example calculates the size of the monthly repayment of a loan of $ 250,000 with a 6% fixed annual interest rate over a period of 15 years according to the annuity. For this, the Excel function PMT will be used.

Note: The PMT function calculates the payment for a loan based on constant periodic payments and a constant interest rate.
File: Repayment.xlsx
  1. Open the practice file.
  2. Format the cells as shown in the figure.
  3. Select cell B4 and press the button Insert Function on the formula bar.
  4. Select in the Financial category the function PMT and click OK.
    The Function Arguments dialog box appears.
  5. Put the cursor in the Rate box and enter B2/12. Then put the cursor in the Nper box and enter B3*12. Finally put the cursor in the Pv box and enter B1.
    Important: Because the period in this example is in months and not in years, the annual rate must be divided by 12 and the number of years must be multiplied by 12.
  6. Click OK.

    The answer -2109.64 appears. Because the result is an amount to be paid, so a debt, Excel displays this as a negative number. That is, in a red font and with a minus sign.

  7. Change this in a positive number by putting a minus sign after the = sign.
    =-PMT(B2/12,B3*12,B1)
