Task: Autosum

For the most common add operation, you can use the Autosum button on the ribbon.

For entering most functions you use the function wizard. This requires several mouse clicks. Because add is the most used operation, Excel has a separate button on the ribbon for this, in the group Editing (tab Home), called Autosum. Using these buttons speeds up entering the add function. Through the triangle arrow, you get access to the functions Average, Count Numbers, Max en Min. In this way, these functions are also quickly accessible.

  1. Start with a new worksheet.
  2. Enter in the cells A1:A6 in succession the following numbers: 120, 121, 122, 123, 124, and 125.
  3. Select cell A7 and then click on Home > button Autosum (group Editing).
    Excel places a selection border around the cells that you probably want to sum.
    Attention: If this is not the area you want to sum, then select with the mouse the correct area.
  4. Confirm the selection by one of the following actions:
    • Click again on the Autosum button.
    • Click on the Enter button at the beginning of the formula bar.
    • Press the Enter key on the keyboard.
    In cell A7 the answer 735 is displayed. The formula in cell A7 is =SUM(A1:A6). This formula is shorter and clearer, therefore better than the formula =A1+A2+A3+A4+A5+A6.
