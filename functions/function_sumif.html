Task: SUMIF

A task in which you learn how you can add numbers that meet certain conditions.

The SUMIF function can be used to add numbers under certain conditions. The spreadsheet has a list of the yields of different coffee types per month and per region. You need to determine the total yields per region.

File: Coffee.xlsx
  1. Open the practice file.
  2. Select cell H2 and press the button Insert Function on the formula bar.
  3. Select in the All category the function SUMIF and click on OK.
    The Function Arguments dialog box appears.
  4. Put the cursor in the Range box and select column C. Then put the cursor in the Criteria box and select cell G2. Finally, put the cursor in the Sum_range box and select column D.
  5. Click OK.
    The total yield for region North 743140 now appears in cell H2.
  6. Select cell H2 and drag down the fill handle to H5.
  7. Format the numbers in H2:H5 as currencies.
Tip: An addition in this example is easier to create with pivots.
