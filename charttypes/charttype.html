        The basic types of charts.

      • Task: Column Chart

        A chart type that is very suitable for displaying time series.

      • Task: Bar Chart

        Chart type that is very suitable for displaying rankings.

      • Task: Pie Chart

        A pie chart for showing the percentage distribution within one data series (group).

      • Task: Line Chart

        You use a line chart to visualize a trend in data over intervals of time.

      • Task: Scatter (XY) Chart

        A scatter chart (XY diagram) is very suitable if you want to analyze and show the relationship between two numeric variables. It also allows you to determine a trend line.

      • Line or Scatter Chart?

        The differences between a line chart and an xy (scatter) chart.

      • Task: Doughnut Chart

        A doughnut chart can be used if you want to show the percentage distribution of multiple data series.

      • Task: Area Chart (stacked)

        You use an area chart to plot multiple data series over a period of time. If the sum of the values is also important then use a stacked area chart.

      • Task: Bubble Chart

        A bubble chart can be used if you want to display the relationship between three numeric variables.

      • Task: Radar Chart

        A radar chart (spider chart) can be used if you want to plot multiple categories among separate axes.
