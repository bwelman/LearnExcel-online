Task: Pie Chart

A pie chart, showing the percentage distribution within one data series (group).

Pie charts are mainly used for displaying the percentage distribution of data within a group, such as election results. Too much information can make a pie chart cluttered. To keep it clear you should have at most 6 or 7 sectors in a pie chart.

Problem

150 pupils have been asked what their favorite pet is. In the table below you will see the results.

Pet Frequency
Cats 70
Dogs 38
Birds 22
Fish 13
Others 7

The data should be displayed in a pie chart, as shown in the following figure.

File: Chart_Pie.xlsx
  1. Open the practice file.
  2. Select any cell in the data area.
  3. ChooseInsert > Insert Pie or Doughnut Chart (group Charts) > Pie.
  4. Add data labels, position Outside End
    The frequency of each pet is displayed at the outside of each piece.
  5. Select the data labels, right-click and select Format Data Labels.
    The task panel Format Data Labels is displayed.
  6. Select Category Name and Percentage. Deselect Value.
  7. Remove legend.
  8. Change chart title in Favorite pet of 150 pupils.
Tip: By choosing to add data labels as Data Call Out instead of Outside End, you get immediately the category names and percentages. Try it.
