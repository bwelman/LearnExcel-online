Language-dependent number formats

Separators for decimals and thousands are dependent on the language of Windows but can be changed in Excel.

The way Excel is formatting numbers depends on the language version of Windows. Especially for the decimal and thousands separators, this may cause confusion and problems. By default, Windows uses the following characters.

  • Dutch
    • Decimal separator comma: 12,34
    • Thousands separator point: 12.345
  • English
    • Decimal separator point: 12.34
    • Thousands separator comma: 12,345

When you want to change these settings permanently, it must be done through the regional settings in the control panel of Windows.

Fortunately, you can temporarily change these settings in Excel. To do this, follow these steps.

  1. Click File > Options > Advanced
  2. Under Editing options clear the checkbox Use system separators.
  3. Type new separators in the Decimal separator and Thousands separator boxes.
Tip: When you want to use the system separators again, select the Use system separators checkbox.
