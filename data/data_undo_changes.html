Undo changes

About how to undo changes.

From almost every action you perform in Excel it is possible to undo it. This is done using the buttons Undo () or Redo (). These buttons are located on the Quick Access Toolbar.

The button Undo reverses the last action. By clicking this button again you will undo the previously performed task. Besides this button there can be an arrowhead, clicking on it shows a list of the most recent actions. By clicking on an action in this list, all changes up to and including this action will be undone.

Using the button Redo can redo the actions you undid. Besides this button there can be an arrowhead with a list of the most recent actions.
