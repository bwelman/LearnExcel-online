Task: Data entry

A simple exercise in data entry.

  1. Select cell A1.
  2. Type home and press Enter.
    The word home is left aligned in the cell and the cell pointer has moved one cell down so now cell A2 is the active cell.
  3. Type Sales 1e quarter and press Enter.
    The text is left aligned in the cell, does not fit within the cell boundaries and therefore seems partly to be in B2, which is not true. It only looks like that because cell B2 is empty. The active cell is now A3.
  4. Type 89 and press Enter.
    The number is right aligned in cell A3. The active cell is now A4.
  5. Type 12.34 and press Enter.
    The number is right aligned in the cell A4. The active cell is now A5.
  6. Type 5,6 and press Enter.
    The content of the cell is left aligned. This is because the comma between 5 and 6. Excel interprets the entry as text and therefore is left-aligned. The active cell is A6.
  7. Type 1/2/2011 and press Enter.
    The date is right aligned in the cell. That's because Excel interprets dates as numbers, treats them as numbers, but gives them a special date format. The active cell is now A7 .
  8. Type =2+3 and press Enter.
    The cell now displays the number 5. That's because Excel treats all data that starts with a = a formula, calculates the result of the formula and displays this result. The active cell is now A8.
    Note: The content of the cell is always the formula and not the result.
  9. Select cell B2, type 10% and press Enter.
    In cell B2 the contents appears right aligned. That's because of the percent sign Excel has interpreted the data as a number. The real content of this cell is therefore 0.1 and this content is displayed as a percentage. And because cell B2 is no longer empty, the text in cell A2 is not fully shown.
  10. Select cell A2.
    In the formula bar you can see the fully entered text is still in the cell.
    Attention: It is possible to adjust the column width so that all the text in cell A2 will be displayed.

In this data entry exercise you have learned the following:

  • By default, text is left aligned.
  • By default, numbers are right aligned.
  • There is an important difference between commas and points in numbers.
  • Formulas always start with the = sign.
  • An input of a date will be treated as a number and formatted as a date.
  • An input of a number with a percent sign will be treated as number and formatted as a percentage. The cell content is the hundredth part of the number entered.
  • A cell has a content and a format. What is displayed is not always the real content.
