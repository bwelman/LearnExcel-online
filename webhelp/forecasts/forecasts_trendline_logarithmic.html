Task: Logarithmic trend line

A logarithmic trend line is best appropriate curved line that fits well with data that in the beginning soon rise (or fall) and then stabilize.
Attention: There should be no null values in the data set.

The equation of the trend line is: y = a*ln(x) + b with a and b as constants.

The following example is about the growth of the number of bacteria in culture.

File: Trendlines.xlsx
  1. Open the practice file and select worksheet Logarithmic
  2. Select a cell in the data area.
  3. Insert a XY (Scatter) Chart.
  4. Select the data serie in the chart by clicking on a point. Right mouse click and choose Add Trend line.
  5. Select at Trendline Options: Logaritmisch. Let equation and R-squared value be displayed in the chart.
  6. Change the chart title in Growth bacteria.
  7. Add Axis titles and change title Y-axis in Number (x million) and title X-axis in Day.
  8. Drag in the chart the equation to an empty place.

R2 = 0.9622 which means that the logarithmic trend line fits the data very well, a fast increase in the beginning followed by a gradual flattening.

Worksheet functions

Equation: y = a*LN(x) + b
  • a =INDEX(LINEST(known_ys ; LN(known_xs)) ; 1 ; 1)
  • b =INDEX(LINEST(known_ys ; LN(known_xs)) ; 1 ; 2)
