Task: Import text/csv

A lot of data in external sources is available as text (filetype .txt) or as comma (or character) separated values (filetype .csv).

File: SalesJan2009.csv
  1. Open a new blank workbook.
  2. Choose tab Data > From Text/CSV (group Get & Transform Data).
  3. Browse to the practice file and click Import.

    A preview of the data will be displayed.

  4. Click Edit.

    The Query Editor is started in a new window. On the right site below Query Settings under Applied Steps you see the steps already applied.

  5. Select the columns Account_Created, Last_Login, Latitude, Longitude) by clicking on the column header and holding the CTRL key. Then choose Remove Columns (group Manage Columns) > Remove Columns.
    An alternative is right mouseclick and then select trhe appropriate action.
    The columns are removed and this step is added to the Applied Steps.

In each column header is on the left site an icon that indicates the applied data type. When you select a column you see the data type also on the ribbon. And this is also the place where you can change the data type.

Price has data type Whole Number, all other columns have data type Text.

  1. Select the first column Transaction_date and then Data Type (group Transform) > Date/Time. When a dialog box appears to ask if the existing conversion needs to be replaced, then choose to replace the current conversion.

In column Name most names started with a capital, but not all.

  1. Select column Name, Right Click and select Transform > Capitalize Each Word.
  2. Choose Close & Load (group Close) > Close & Load.

    The data is loaded in an Excel table in a new sheet. You can work with the data and save the the Excel map for later use.

    Note: It is always possible to change the query afterwards. For this select a cell in the table and choose tab Query > Edit (group Edit). From here you can also refresh your data, all steps will then repeated.
