Get & Transform data

To use Get & Transform in Excel 2016, you actually create a query in your workbook. A query enables you to connect to, preview, and transform data from a wide variety of available data sources.

The first step is making a connection to a data source. When the connection is established you get a preview of the data. Then you can add steps to shape and transform the data. To do this, a Query Editor is started to create a set of instructions that carry out your commands. The last step is loading the data as a table in a worksheet.

Note: You also have the possibility to load the data into the built-in Data Model.

You can also edit the query later and/or refresh the data.

To create a query, use the Data tab in the ribbon, then select one of the actions in the Get & Transform group.

There are buttons for getting data from text/csv, web and an Excel Table/range. These data sources and a lot more are available through the Get Data button.

The group Queries & Connections has a button for refreshing queries and a button for viewing and managing existing queries and connections.
