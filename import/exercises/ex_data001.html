Exercise: data001 - Import simple csv file

Level: 2

File: Revenues.csv

First open this file with a plain text editor (Notepad) to investigate how the file looks like. Important to know is if there are field names in it and which separator symbol (comma, semicolon, ...) is used. Mostly Excel recognize this automatically, but it is useful to know this yourself.

Import the data throughGet & Transform. Use also the query editor to verify that the fields are converted to the correct data type. If not, change the data type.
