Importing data

In practice, a lot of data is coming from external sources: text/csv, web, excel files, databases, ... To analyze these data in Excel they must be imported first and sometimes transformed and restructured. Excel 2016 has integrated the Power Query technology under the Get & Transform group in the Data ribbon. With this technology you can make connections to several data sources, shape that data (for example remove a column, change a data type) or transform the data. And all your steps are collected in a query script which lets you undo, redo or modify any step. With this script, you can repeatable refresh the import of the data.

Note: These features are also present in Microsoft Power BI and in the Power Query Add-In for previous versions of Excel.
