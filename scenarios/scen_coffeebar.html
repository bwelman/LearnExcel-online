Task: Coffee bar "The Brown Bean"

A task in working with scenario management.

In the following figure, you see a calculation model of a coffee bar. The gray cells are the variable cells.

In addition to the current situation, the owner of the coffee bar wants to explore two other scenarios to see the effect on the profit.

  • A scenario with 10% higher prices, salaries less than $ 30000, and advertising less than $ 5000.
  • A scenario with 5% higher prices, 5% lower order costs, and $ 2000 more advertising.
File: BrownBean.xlsx
  1. Open the practice file.
Study the model. Identify the changing cells and the result cell. Study the used formulas and see that all cells have names. If you give meaningful names to the cells then these names will be used in the summary report instead of the addresses. Then it will be easy to read the report.
  1. Choose tab Data > What-If Analysis (group Forecast) > Scenario Manager.
  2. Click on Add.
    The dialog box Edit Scenario appears.
  3. Type in the box Scenario name the text Current. Put the cursor in the box Changing cells and then select in the worksheet the ranges B4:B6 and B11:B14. Finally, put the cursor in the text box Comment and type a brief explanation of the scenario, for example, Current situation.
    Tip: Use the CTRL key to select multiple ranges.
  4. Click OK.
  5. Click on Add for entering a new scenario.
  6. Give the scenario the name Higher prices, less salary and advertising and give it a brief description and click OK.

    The dialog box Scenario Values appears displaying the current values of the changing cells.

  7. Now type the new values in the cells that change for this scenario.
    Instead of calculating these values yourself, you can enter a formula. Excel automatically calculates the new value.
    Yield_per_Order=1.1*2.5
    Salaries=300000-30000
    Advertisement=20000-5000
  8. Click OK.

    A warning is displayed:

    The entered formulas are replaced by the results of the formulas.
  9. Click OK.
  10. The same way, create another scenario with the name Higher prices, less order costs and more advertising and adjust the values for this scenario.
  11. Select a scenario and click Show.
    In the worksheet, the values of the changing cells and the dependent cells will be adapted for the selected scenario.
  12. Click on Summary to create a report.
  13. Click OK.

    A new worksheet with a summary report is created.
