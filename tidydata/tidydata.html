Tidy Data

When importing data from external sources, you don't have control over the format and type of data and the way it is organized. Before you can analyze the data, you often have to spend a lot of time to clean and structure the data.

Many data files are dirty, data is missing or the data is not well structured. Excel has some features to help you with this job. But unfortunately Excel can't track and log your manipulations. So it is important to document what you have done with the original data to make it suitable for analysis. Actions that you perform with the Power Query-Editor are recorded in a script. Therefore try to use the Power-Query Editor as much as possible.

In this chapter a number of tasks are addressed to create tidy data.
