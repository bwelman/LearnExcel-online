Tips Goal Seek

Some tips if Goal Seek does not produce the right answer.

Goal Seek does not produce the right answer

It is possible that there is no answer or that Excel cannot find one. In that case a dialog box pops up.

If the solution exists, but Excel can not find it, then check the following possible solutions:

  1. Check in the model that the result cell depends on the changing cell. The result cell should always contain a formula or a function. And the changing cell should only contain a value and no formula or function.
  2. Choose another initial value for the changing cell.
  3. Increase the maximum number of iterations (default value is 100) with File > Options > Formulas
