Mouse selections

An overview of the options to select cells with the mouse.

It is very common that you want to perform one command on a number of cells. For example, if you want to centralize the text in a number of cells, or change the font of these cells. Therefore it is very important that you get a good routine selecting multiple cells at the same time. In the table below you can see the most common selections.

desired selection method
single cell Click on the cell.
range of cells Click the first cell in the range and then drag to the last cell.
single column Click on the column letter.
adjacent columns Click on the first column letter and drag to the last column letter.
single row Click on the row number.
adjacent rows Click on the first row number and drag to the last row number.
all cells in worksheet Click on the Select All button, above row number 1 and left from column A.
Tip: Shortcut CTRL A

When selecting, you can also use the Shift key or the CTRL key, with the following effects:

SHIFT key
Using the SHIFT key you can select an adjacent area.
  • For a rectangular area, click on one of the corner cells, hold down the SHIFT key and click on the cell in the diagonally opposite corner.
  • For adjacent columns, select the first column, hold down the SHIFT key while selecting the last column.
  • For adjacent rows, select the first row, hold down the SHIFT key while you select the last row.
CTRL key
Using the CTRL key, you can select non-adjacent area's. Select the first cell, area, row or column, hold down the CTRL key while you select the other cells, areas, rows and columns.
Attention:
  • Selections are only possible when the mouse pointer has the shape for making selections (thick plus sign): .
  • A selected area is highlighted in color except the cell where you started the selection. This cell is the active cell and has a white background.
Tip: Dragging from bottom to top and from right to left is easier then doing it in the reverse direction.

Indicating areas

You can use cell addresses for indicating adjacent areas. A few examples:

  • A2: C7 is a rectangular area cells of A2 through C7. You always start with the cell address of the upper left corner, followed by a colon and then the address of the bottom right corner.
  • B:E are the columns B through E.
  • 3:9 are the rows 3 through 9.
